<p>Cross-polarisation can greatly improve the image quality and thus the resulting 3D scans. Therefore, we have developed the <a href="https://openscan.eu/products/polarizer-module-for-openscan-classic" title="Polarizer Module for OpenScan Classic">polarizer module</a> for the <a href="https://openscan.eu/products/openscan-classic">OpenScan Classic</a>, that does exactly that.</p>
<p><img alt="" src="/assets/img/posts/2025-02-04-polarizer-module-for-openscan-classic/comparison.png"/><em>Left: without polarization --&gt; lot of reflections, right: with cross-polarization</em></p>
<p>You can either get the polarizer module in our <a href="https://openscan.eu/products/polarizer-module-for-openscan-classic">online shop</a>, or download and print the files from <a href="https://github.com/OpenScan-org/OpenScan-Design/tree/main/files/Classic/V1">GitHub</a>. Note, that this module is compatible with the OpenScan Classic only.</p>
<p>In order to build the module yourself, you will need translucent filament for the main part, black filament for the inner, smaller part and <a href="https://openscan.eu/products/polarisationsfilterfolie">linear polarizer film</a>.</p>
