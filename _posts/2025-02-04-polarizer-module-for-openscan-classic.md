Cross-polarisation can greatly improve the image quality and thus the resulting 3D scans. Therefore, we have developed the [polarizer module](https://openscan.eu/products/polarizer-module-for-openscan-classic) for the [OpenScan Classic](https://openscan.eu/products/openscan-classic), that does exactly that.

![](/assets/img/posts/2025-02-04-polarizer-module-for-openscan-classic/comparison.png)*Left: without polarization --> lot of reflections, right: with cross-polarization*

You can either get the polarizer module in our [online shop](https://openscan.eu/products/polarizer-module-for-openscan-classic), or download and print the files from [GitHub](https://github.com/OpenScan-org/OpenScan-Design/tree/main/files/Classic/V1). Note, that this module is compatible with the OpenScan Classic only.

In order to build the module yourself, you will need translucent filament for the main part, black filament for the inner, smaller part and [linear polarizer film](https://openscan.eu/products/polarisationsfilterfolie).
